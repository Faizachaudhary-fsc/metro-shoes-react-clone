Metro Shoes Website | MERN Stack Project
Built a full-stack e-commerce website for Metro Shoes using the MERN stack.

 🔹 Backend: Node.js, Express.js

Implemented responsive UI, smooth navigation, and 
complete frontend and backend integration.
